# Monigue DEMO Font

## How to Add the Font

1. **Download the font:**
   - Visit: https://exfont.com/moniguedemo-regular.font
   - Or: https://www.dafont.com/monigue-demo.font
   - Download the font file (usually a ZIP)

2. **Extract and place the font file:**
   - Extract the downloaded ZIP file
   - Find the font file (.ttf, .otf, or .woff format)
   - Rename it to one of these (place it in this `fonts` folder):
     - `MonigueDEMO.woff2` (preferred)
     - `MonigueDEMO.woff`
     - `MonigueDEMO.ttf`
     - `MonigueDEMO.otf`

3. **That's it!** The font will automatically load when you run the app.

## Quick Setup (Windows PowerShell)

Run the download script:
```powershell
.\download-font.ps1
```
